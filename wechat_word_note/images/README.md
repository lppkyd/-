# TabBar 图标说明

由于无法在此环境创建实际图片文件，请按以下步骤添加 TabBar 图标：

## 需要的图标文件

在 `images/` 目录下需要以下 6 个图标文件：

1. `typing.png` - 打字图标（未选中）
2. `typing-active.png` - 打字图标（选中）
3. `gallery.png` - 词典图标（未选中）
4. `gallery-active.png` - 词典图标（选中）
5. `errors.png` - 错题本图标（未选中）
6. `errors-active.png` - 错题本图标（选中）

## 图标规格

- 尺寸：81px × 81px
- 格式：PNG
- 大小：< 40KB

## 临时解决方案

在开发调试阶段，可以：

1. 使用微信开发者工具的默认图标
2. 或从 iconfont.cn 下载合适的图标
3. 或使用纯色正方形作为占位图标

## 推荐图标

- 打字：键盘图标
- 词典：书本图标
- 错题本：笔记本图标
