# 功能更新: 用户中心与导航重构 (User Center & Refactor)

**日期**: 2026-01-29  
**版本**: v1.3.0  
**开发者**: AI Agent (Antigravity Claude)

---

## 核心变更

### 1. 导航栏重构
底部导航栏的第三个图标从 **"📝 错题本"** 替换为 **"👤 我的"**。
错题本不再作为一级入口，而是移动到了用户中心内部。

**新版导航结构**:
1.  **⌨️ 练习** (`/pages/typing/`)
2.  **📚 词典** (`/pages/gallery/`)
3.  **➕ 添加** (`/pages/add-word/`) - 快速添加生词
4.  **👤 我的** (`/pages/user/`) - 用户中心

### 2. 新增用户中心页面 (`pages/user/`)
包含以下功能入口：
*   **📝 我的错题本**: 点击跳转到原错题本页面。显示当前错题数量。
*   **📖 创建词典**: 快捷跳转到自定义词典创建页。
*   **🗑️ 清除缓存**: 允许用户重置所有本地数据（慎用）。
*   **ℹ️ 关于**: 显示应用版本信息。

### 3. 错题本页面调整 (`pages/errors/`)
*   移除了底部的全局导航栏，使其成为一个纯粹的子页面。
*   用户可以通过点击左上角返回按钮回到用户中心。

### 4. 修复: 预设词典加载问题
*   **问题**: 预设词典（CET4/CET6）此前无法打开或显示为空。
*   **修复**: 调整了 `utils/dict.js` 中的引用路径，优先使用相对路径 `require('../data/cet4_v2.js')`，解决了在小程序环境下的路径解析问题。
