## 🔧 技术要点

### 1. API接口使用

**有道 Suggest API**:
```
接口: https://dict.youdao.com/suggest
方法: GET
参数:
  - q: 查询单词
  - num: 返回数量（1）
  - doctype: json

返回示例:
{
  "result": {"msg": "success", "code": 200},
  "data": {
    "entries": [
      {"explain": "n. 苹果", "entry": "apple"}
    ]
  }
}
```

### 2. 微信小程序网络请求

**域名配置**:
- `dict.youdao.com` 已在项目中使用（发音API）
- 无需额外配置域名白名单

**请求代码**:
```javascript
wx.request({
  url: 'https://dict.youdao.com/suggest',
  data: { q: 'apple', num: 1, doctype: 'json' },
  method: 'GET',
  timeout: 5000,
  success: (res) => { /* 处理响应 */ },
  fail: (err) => { /* 错误处理 */ }
})
```

### 3. 错误处理机制

**分类处理**:
- 空输入 → Toast 提示
- 网络错误 → Modal 对话框提示
- 查询无结果 → Modal 提示，允许手动输入
- 超时 → 5秒后自动 reject

## ⚠️ 已知限制

1. **API稳定性**: 使用的是有道非官方API，可能存在调用限制
2. **音标缺失**: 免费 Suggest API 不返回音标信息
3. **详细释义**: 仅返回简单释义，无例句、词根等高级信息
4. **调用频率**: 无明确的QPS限制文档，建议适度使用

**解决方案**: 参见 `doc/FUTURE_ENHANCEMENTS.md` 中的付费API升级计划
