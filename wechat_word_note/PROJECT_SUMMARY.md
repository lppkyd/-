## 🚀 如何预览和调试

### 快速开始（3 步）

1. **下载微信开发者工具**

   - 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **导入项目**

   - 打开微信开发者工具
   - 选择「导入项目」
   - 项目目录：`qwerty-learner-mini`
   - AppID：选择「测试号」

3. **开始调试**
   - 自动编译后即可在模拟器中预览
   - 点击「预览」扫码在真机测试

### 重要配置

**启用发音功能**（必须）：

```
右上角「详情」→「本地设置」→ 勾选「不校验合法域名...」
```

### 详细指南

请查看 `QUICK_START.md` 获取完整的预览和调试指南。

## ⚠️ 注意事项

### 1. 图标问题

- TabBar 图标文件未包含（需自行添加）
- 临时方案：使用 emoji 作为导航图标
- 永久方案：添加 PNG 图标到 `images/` 目录

### 2. 网络依赖

- 发音功能依赖有道词典 API
- 需要配置请求域名或勾选「不校验」
- 离线模式下发音功能不可用

### 3. 词典大小

- 当前两个词典约 1MB
- 小程序主包限制 2MB
- 如需更多词典，建议使用分包加载

### 4. 性能优化

- 首次加载词典数据可能较慢
- 建议添加加载状态提示
- 可考虑按章节懒加载
